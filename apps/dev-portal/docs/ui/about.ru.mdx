Condo UI – это библиотека адаптивных React-компонентов, позволяющая создавать мини-приложения,
интерфейс которых будет схожим с интерфейсом Condo.

## Интерактивная песочница

Все компоненты Condo UI можно найти в нашей [интерактивной песочнице](https://condo.d.doma.ai/ui/index.html),
где можно посмотреть на их поведение, параметры и состояния (для отображения панели управления используйте клавишу `A`).

## Установка

Чтобы установить библиотеку, просто выполните следующую команду в соответствии с вашим пакетным менеджером:
<Tabs labels={['npm', 'yarn', 'pnpm']}>
    ```bash
    npm i @open-condo/ui
    ```
    ```bash
    yarn add @open-condo/ui
    ```
    ```bash
    pnpm i @open-condo/ui
    ```
</Tabs>

После чего добавьте CSS-стили Condo UI в свое приложение и все компоненты Condo UI станут доступны для использования:
```typescript
import '@open-condo/ui/dist/styles.min.css'
```


## Использование библиотеки

Вы можете импортировать необходимый компонент следующим образом:

```typescript jsx
import { Button } from '@open-condo/ui'
import type { ButtonProps } from '@open-condo/ui'
```

Помимо компонентов в библиотеке так же доступна и вся цветовая палитра Condo UI. Получить её можно либо в формате JS-объекта,
либо в формате CSS-переменных вместе с другими CSS-токенами:
<Tabs labels={['JS-object', 'CSS-vars']}>
    ```typescript
    import { colors } from '@open-condo/ui/colors'
    ```
    ```typescript
    import '@open-condo/ui/style-vars/css'
    ```
</Tabs>
