Привет! Добро пожаловать в документацию для разработчиков платформы Condo! 
Наша платформа предоставляет инструменты для управления многоквартирными домами и жилыми комплексами, улучшая взаимодействие между управляющими организациями, их сотрудниками и жителями. 

Вы можете выбрать один из следующих способов интеграции, в зависимости от ваших потребностей и целей:
