## О версии данных

Во все методах API мы используем параметр `dv` - версия отправляемых данных.
Данный параметр позволяет нам поддерживать обратную совместимость API в случаях,
когда тот или иной метод значительно меняет свою сигнатуру или сигнатуру возвращаемого значения.
В таком случае мы сможем определить версию API, используемого клиентом по значению параметра `dv`

<Alert type={'info'}>
    На текущий момент ни одна наша модель не терпела значительных изменений,
    поэтому в своих запросах всегда используйте `dv = 1` в качестве значения данного параметра.
</Alert>
