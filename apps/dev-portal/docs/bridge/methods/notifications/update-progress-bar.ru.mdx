Данный метод позволяет обновить созданное ранее [уведомление со шкалой прогресса](/docs/bridge/methods/notifications/show-progress-bar).

## Платформы
- B2B-Web


## Параметры метода

| Имя параметра  | Тип параметра                                     | Обязательный | Описание параметра                                                                                                                                                                                      |
|----------------|---------------------------------------------------|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| barId          | string                                            | да           | ID уведомления                                                                                                                                                                                          |
| data           | JSON                                              | да           | Данные для обновления. Обновлено может быть любое поле из тех, что [указаны в создании уведомления](/docs/bridge/methods/notifications/show-progress-bar#method-params), а также числовое поле progress |

## Возвращаемое значение

При успешном выполнении метод вернет назад JSON-объект, содержащий единственное поле `updated`:
```json
{ "updated": true }
```

## Пример использования

```typescript
import bridge from '@open-condo/bridge'
// Update progress
bridge.send('CondoWebAppUpdateProgressBar', {
    barId: '3f4714ab-30f9-4bd4-b91b-67e9c5b7f1a8',
    data: {
        progress: 20,
        description: '5 seconds left'
    },
})

// Finish task
bridge.send('CondoWebAppUpdateProgressBar', {
    barId: '3f4714ab-30f9-4bd4-b91b-67e9c5b7f1a8',
    data: {
        progress: 100,
        description: 'done'
    },
})
```
