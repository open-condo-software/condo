Данный метод позволяет показать пользователю уведомление.
Наиболее частый пример использования - сообщения об произошедшей ошибке / успехе.

## Платформы
- B2B-Web


## Параметры метода

| Имя параметра | Тип параметра                                     | Обязательный | Описание параметра                                                        |
|---------------|---------------------------------------------------|--------------|---------------------------------------------------------------------------|
| type          | info  \|  warning  \|  error  \|  success         | да           | Тип показываемого уведомления                                             |
| message       | string                                            | да           | Заголовок уведомления. Пример: "Произошла ошибка"                         |
| description   | string                                            | нет          | Тело уведомления, в котором стоит описать основную информацию уведомления |

## Возвращаемое значение

При успешном выполнении метод вернет назад JSON-объект, содержащий единственное поле `success`:
```json
{ "success": true }
```

## Примеры использования
```typescript
import bridge from '@open-condo/bridge'

bridge.send('CondoWebAppShowNotification', {
    type: 'error',
    message: 'Error occurred during integration',
    description: 'Your account does not have the email address required to connect the mini-application'
})
```
