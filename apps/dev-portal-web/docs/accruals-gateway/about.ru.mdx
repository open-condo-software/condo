**Шлюз начислений** — агрегирующий API-сервис, предоставляющий единый интерфейс для доступа к данным о начислениях за жилищно-коммунальные услуги от множества поставщиков.

## API Endpoints

Используйте следующие базовые URL в зависимости от окружения:

### Production
```
https://accruals-gateway.doma.ai
```

### Development
```
https://accruals-gateway.d.doma.ai
```

## Основные функции

### [Поиск поставщиков →](/docs/accruals-gateway/search)
- Поиск организаций по ИНН, названию или другим реквизитам

### [Получение начислений →](/docs/accruals-gateway/check)
- Доступ к данным о начислениях по лицевым счетам
- Получение структурированных данных в JSON-формате
- Ссылки на оригинальные PDF-документы (квитанции)

### [Управление платежами →](/docs/accruals-gateway/pay)
- Передача информации об оплатах
- Подтверждение успешных транзакций
- Синхронизация статусов платежей
