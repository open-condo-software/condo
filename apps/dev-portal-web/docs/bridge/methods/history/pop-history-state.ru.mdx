Данный метод убирает одну или несколько записей из стека навигации мини-приложения — аналог `window.history.go(-n)`.

После вызова хост отправит событие [`CondoWebAppHistoryPopState`](/docs/bridge/methods/history/events) с состоянием, которое оказалось на вершине стека.

<Alert type={'warning'}>
   В отличие от `window.history.go`, навигация в мини-приложениях не позволяет передвигаться «Вперёд». Учтите это при проектировании логики мини-приложения
</Alert>

## Платформы
<SupportedPlatforms platforms={['B2CWeb']} />

## Параметры метода

| Имя параметра | Тип параметра | Обязательный | Описание параметра                                                                       |
|---------------|---------------|--------------|------------------------------------------------------------------------------------------|
| amount        | number        | нет          | Количество записей для удаления из стека. По умолчанию `1`                               |

## Возвращаемое значение

При успешном выполнении метод вернет назад JSON-объект, содержащий единственное поле `success`:
```json
{ "success": true }
```

## Пример использования

```typescript
import bridge from '@open-condo/bridge'

await bridge.send('CondoWebAppPopHistoryState')
```
