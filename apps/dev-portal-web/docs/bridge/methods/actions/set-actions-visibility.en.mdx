This method allows you to show or hide the actions block without resending the actions list.

## Platforms
- B2B-Web
- B2C-Mobile

## Method params

| Parameter name | Parameter type | Required | Parameter description |
|---|---|---|---|
| visible | boolean | yes | Whether the host should display the actions UI |

## Return value

If successful, the method will return a JSON object containing a single `success` field:
```json
{ "success": true }
```

## Usage examples

Hide actions:
```typescript
import bridge from '@open-condo/bridge'

bridge.send('CondoWebAppSetActionsVisibility', { visible: false })
```

Show actions again (after you have set actions config at least once):
```typescript
import bridge from '@open-condo/bridge'

bridge.send('CondoWebAppSetActionsVisibility', { visible: true })
```

<Alert type='info'>
    You still need to call `CondoWebAppSetActionsConfig` once to provide the actions list.
</Alert>
