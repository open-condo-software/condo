This event is sent by the host to notify your mini-app that a user clicked an action button.

## Platforms
- B2B-Web
- B2C-Mobile

## Method params

| Parameter name | Parameter type | Required | Parameter description |
|---|---|---|---|
| actionId | string | yes | Identifier of the clicked action (generated by host) |

## Usage examples

### Recommended: use the helper hook

```typescript jsx
import React, { useMemo } from 'react'

import bridge from '@open-condo/bridge'
import { useActionHandlers } from '@open-condo/miniapp-utils'

export function Example (): React.ReactNode {
    const handlers = useMemo(() => ({
        'action-id-1': () => {
            // handle save
        },
        'action-id-2': () => {
            // handle download
        },
    }), [])

    useActionHandlers(bridge, handlers)

    return null
}
```

<Alert type='info'>
    The `actionId` comes from the `actionsIds` array returned by `CondoWebAppSetActionsConfig`.
</Alert>

### Low-level subscription to event without the helper hook

```typescript
import bridge from '@open-condo/bridge'

const listener = (event) => {
    if (event.type !== 'CondoWebAppSendActionId') return

    const { actionId } = event.data
    if (actionId === 'your_action_id') {
        // handle your button's onClick logic
    }
}

bridge.subscribe(listener)
```
