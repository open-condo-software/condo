Этот метод позволяет обновить конкретное действие (кнопку) по его id.

## Платформы
- B2B-Web
- B2C-Mobile

## Параметры метода

| Название параметра | Тип параметра | Обязательный | Описание |
|---|---|---|---|
| id | string | да | Идентификатор действия (используйте id из `CondoWebAppSetActionsConfig`) |
| params | ActionPatch | да | Частичное обновление параметров действия |

### Параметры ActionPatch (button)

| Название параметра | Тип параметра | Обязательный | Описание |
|---|---|---|---|
| label | string | нет | Текст кнопки |
| type | 'primary' \| 'secondary' \| 'accent' | нет | Визуальный тип кнопки |
| visible | boolean | нет | Видимость кнопки |
| disabled | boolean | нет | Отключить кнопку |
| danger | boolean | нет | Опасный стиль кнопки |
| loading | boolean | нет | Состояние загрузки |
| icon | 'download' | нет | Иконка |
| iconSize | 'auto' \| 'large' \| 'medium' \| 'small' | нет | Размер иконки |
| size | 'medium' \| 'large' | нет | Размер кнопки |
| compact | boolean | нет | Компактный стиль |
| minimal | boolean | нет | Минимальный стиль |

## Возвращаемое значение

При успехе метод вернет JSON объект с единственным полем `success`:
```json
{ "success": true }
```

## Пример использования

```typescript
import bridge from '@open-condo/bridge'

bridge.send('CondoWebAppUpdateActionConfig', {
    id: 'action-uuid',
    params: {
        loading: true,
    },
})
```
