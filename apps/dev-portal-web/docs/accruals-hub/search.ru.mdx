`GET /api/search`

Ищет организации по части названия или по ИНН.

## 🔐 Авторизация

[Посмотреть документацию по авторизации →](/docs/accruals-hub/auth)

## Query parameters

| Поле    | Тип/Формат                     | Обязательный | Описание                                                   | Пример                    |
|---------|--------------------------------|--------------|------------------------------------------------------------|---------------------------|
| search	 | string\<от 3 до 100 символов\> | ✅            | Поисковая строка часть названия организации или полный ИНН | `9731062087`, `Столичный` |

## Пример запроса

```
GET /api/search?search=9731062087
```

## Ответ

| Параметр     | Тип/Формат     | Обязательный | Описание                              | Пример                                         |
|--------------|----------------|--------------|---------------------------------------|------------------------------------------------|
| id           | string\<uuid\> | ✅            | Уникальный идентификатор организации  | `452f081a-7612-48e1-bc4e-89d6cea366e7`         |
| organization | string         | ✅            | Наименование организации              | `ООО УК "Столичный дом"`                       |
| categories   | string         | ✅            | Категории услуг, разделенные запятыми | `Квартплата`,`Водоснабжение, Электроснабжение` |
| tin          | string         | ✅            | ИНН организации                       | `7712345678`                                   |
| bankAccount  | string         | ✅            | Расчетный счет организации            | `40702810500000001234`                         |

## Пример ответа

```json
{
    "data": [
        {
            "id": "452f081a-7612-48e1-bc4e-89d6cea366e7",
            "organization": "ООО УК \"Столичный дом\"",
            "categories": "Квартплата",
            "tin": "7712345678",
            "bankAccount": "40702810500000001234"
        }
    ]
}
```

## Коды ошибок

| Код ошибки               | HTTP статус | Сообщение                 | Причина                     | Решение                                     |
|--------------------------|-------------|---------------------------|-----------------------------|---------------------------------------------|
| `VALIDATION_FAILED`      | 400         | Invalid format of input   | Ошибка валидации параметров | Проверьте формат вводимых данных            |
| `ORGANIZATION_NOT_FOUND` | 404         | Organization is not found | Организация не найдена      | Уточните поисковый запрос                   |
| `INTERNAL_SERVER_ERROR`  | 500         | Internal server error     | Внутренняя ошибка сервера   | Попробуйте позже или обратитесь в поддержку |
