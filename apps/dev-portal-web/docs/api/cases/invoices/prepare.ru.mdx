## Общая информация
У управляющих компаний есть потребность продавать своим жителям услуги и товары.
Например, сотрудники УК могут оказывать услуги по ремонту коммуникаций и сетей, замене и подключению оборудования.
Чтобы жители могли видеть список услуг, в Condo есть раздел Маркетплейс (https://condo.d.doma.ai/marketplace).
Тут сгруппирован функционал управления витриной и счетами.

## Настройка Маркетплейса
Чтобы пользоваться функционалом Маркетплейса, необходимо выполнить первоначальную настройку (подключить Маркетплейс).
Для этого предусмотрено два шага:
- На первом шаге нужно указать реквизиты для получения оплаты за услуги, выбрать режим налогообложения и ставку НДС.
- На втором шаге нужно указать список почтовых адресов для получения детализации платежей и принять оферту. Принятая оферта будет отправлена на указанную почту.

<Alert type="info">
    Если Маркетплейс не настроен, то будет предложено это сделать.
</Alert>
