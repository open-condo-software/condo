# Miniapp Template

This project was generated by `@open-condo/create-miniapp-cli`.

## Quick start

1. Install dependencies:
   - `yarn`
2. Prepare environment:
   - `yarn prepare`
3. Generate GraphQL types:
   - `yarn maketypes`
4. Start app:
   - `yarn dev`


## Notes

- This is a generic fullstack template (Keystone + Next.js).
- Replace placeholder content and extend domains according to your business logic.
